# Wedding Master

A web application for wedding planning and management.

## Project Structure

The project is divided into two main parts:

### Backend
Server-side implementation that handles data processing and storage.

### Frontend
Client-side implementation that provides the user interface, containing:
- `docs`: Documentation files
- `node_modules`: Node.js dependencies
- `public`: Publicly accessible files
- `src`: Source code
  - `components.json`: Component configurations
  - Various configuration files including:
    - `eslint.config.js`
    - `index.html`
    - `jsdoc.json`
    - `package.json`
    - `postcss.config.js`
    - `tailwind.config.js`
    - `tsconfig` files

## Getting Started

### Prerequisites
- Node.js
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/wedding-master.git
cd wedding-master
```

2. Install dependencies
```bash
# For backend
cd backend
npm install

# For frontend
cd frontend
npm install
```

3. Run the development server
```bash
# For backend
cd backend
npm run dev

# For frontend
cd frontend
npm run dev
```
